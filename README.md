# Blokqo Landing

Landing page experimental do **Blokqo** — uma plataforma global colaborativa que transforma problemas urbanos (buracos, iluminação apagada, calçadas perigosas etc.) em dados úteis para pessoas, empresas e cidades.

O objetivo deste repositório é **validar interesse** antes de investir tempo e dinheiro no desenvolvimento do app.

## O que tem aqui

- Página estática em **HTML/CSS**  
- Formulário integrado ao **Formspree** (`https://formspree.io/f/movbgeyk`)  
- Conteúdo em português, pensado para funcionar para qualquer cidade  
- Estrutura focada em conversão (coleta de e-mail, cidade, perfil e expectativas de uso)
